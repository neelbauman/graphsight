## GraphSight Agent v6：図解デジタル化エンジンの概要

**要旨**
本システムは、手書きや画像のフローチャートを、編集可能なコード（Mermaid形式）へ高精度に自動変換するAIエージェントです。
従来の生成AIが苦手としていた「細部の読み間違い」や「修正時の品質劣化（デグレ）」という課題を、**「構造的差分（Structural Diff）」**という新技術によって解決し、実業務に耐えうる精度と安定性を実現しました。

### 従来の課題とv6の解決策

生成AIによる画像認識は画期的ですが、ビジネス利用においては2つの大きなリスクがありました。

1. **ハルシネーション（嘘）:** 細かい文字や薄い線を勝手に捏造・削除してしまう。
2. **修正の不安定さ:** 「Aを直して」と指示すると、合っていたBまで壊してしまう（デグレ）。

**GraphSight Agent v6**は、人間が慎重に清書を行うプロセス（下書き→自己検閲→部分修正）をアルゴリズムで再現することで、これらのリスクを排除しました。

### コアとなる3つの技術的優位性

#### 1. 構造的な修正プロセス（Structural Diff）

これが最大の特徴です。AIが画像を一度テキスト化した後、修正が必要な箇所を「テキストの書き直し」ではなく、**「データベースへの操作コマンド（追加・削除・変更）」**として処理します。
これにより、**修正対象以外の正しい箇所には一切触れない**ため、修正による品質劣化を100%防ぎます。

#### 2. 人間模倣の視覚確認フロー（Zoom & Enhance）

全体をざっくり見るだけでなく、自信がない箇所（文字が潰れている、線が重なっている等）を自動検出し、**その部分だけを拡大（クロップ）して再確認**します。
さらに、読み取れない場合は画像処理で線を太くするなどの**「鮮明化（エンハンス）」**を行い、執念深く正解を導き出します。

#### 3. データ構造の完全正規化

AIが出力する曖昧なデータを、システム内部で厳密なデータモデル（構造化データ）に変換・管理します。これにより、出力されるコードは常に構文エラーのない、即座に利用可能な状態が保証されます。

### ビジネスインパクト

* **業務効率の劇的向上:** 既存の資料やホワイトボードの写真を、エンジニアが手作業で書き起こす工数をほぼゼロにします。
* **DXの加速:** 画像の中に死蔵されていた業務フローが「検索・編集可能なデータ」になることで、プロセス改善やシステム移行が加速します。

---

**結論**
GraphSight Agent v6は、単なる画像認識ツールではなく、**「認識・検証・修正」を自律的に行う高品質デジタイザ**です。信頼性が求められるドキュメント管理やシステム設計の現場において、即戦力となるクオリティと思っています。
